## üåê Deployment Instructions

### Option 1: GitHub Pages (Free)

1. **Create a GitHub Account** (if you don't have one):
   - Go to [github.com](https://github.com)
   - Sign up for a free account

2. **Create a New Repository**:
   - Click the "+" icon in the top right
   - Select "New repository"
   - Name it: `harsh-it-solutions` (or any name you prefer)
   - Make it public
   - Check "Add a README file"
   - Click "Create repository"

3. **Upload Your Files**:
   - Click "uploading an existing file"
   - Drag and drop all files (`index.html`, `styles.css`, `script.js`)
   - Write a commit message: "Initial website upload"
   - Click "Commit changes"

4. **Enable GitHub Pages**:
   - Go to repository Settings
   - Scroll down to "Pages" section
   - Under "Source", select "Deploy from a branch"
   - Select "main" branch and "/ (root)" folder
   - Click "Save"

5. **Access Your Website**:
   - Your site will be available at: `https://yourusername.github.io/harsh-it-solutions`
   - It may take a few minutes to deploy

### Option 2: Vercel (Free)

1. **Create a Vercel Account**:
   - Go to [vercel.com](https://vercel.com)
   - Sign up with GitHub, GitLab, or Bitbucket

2. **Deploy from GitHub**:
   - Click "New Project"
   - Import your GitHub repository
   - Click "Deploy"
   - Your site will be live in seconds!

3. **Custom Domain** (Optional):
   - Go to project settings
   - Add your custom domain
   - Follow DNS configuration instructions

### Option 3: Netlify (Free)

1. **Create a Netlify Account**:
   - Go to [netlify.com](https://netlify.com)
   - Sign up for free

2. **Deploy via Drag & Drop**:
   - Create a folder with all your files
   - Drag the folder to Netlify's deploy area
   - Your site will be live instantly!

3. **Deploy from Git** (Alternative):
   - Connect your GitHub repository
   - Select the repository
   - Click "Deploy site"

## üé® Customization Guide

### Colors and Branding

Edit the CSS variables in `styles.css`:

```css
:root {
    --primary-color: #2563eb;    /* Main brand color */
    --secondary-color: #1e40af;  /* Secondary brand color */
    --accent-color: #3b82f6;     /* Accent color */
}
```

### Contact Information

Update contact details in `index.html`:

```html

<a href="tel:+1234567890">+1 (555) 123-4567</a>

<!-- Email addresses -->
<a href="mailto:info@harshit.site">info@harshit.site</a>

<!-- WhatsApp link -->
<a href="https://wa.me/15551234567">WhatsApp</a>

<!-- Address -->
```

### Services

Modify services in the `serviceDetails` object in `script.js`:

```javascript
const serviceDetails = {
    repair: {
        title: 'Your Service Title',
        content: 'Your service description and details'
    }
    // Add more services as needed
};
```

### Images

Replace image URLs in `index.html` with your own:

```html
<!-- Use your own images from Unsplash or upload your own -->
<img src="https://your-image-url.com/image.jpg" alt="Description">
```
